# Jose's ZSH Theme
A simple theme for *oh-my-zsh*, to install move the `jose.zsh-theme` file to `~/.oh-my-zsh/themes/`. Work inspired by: [oh-my-zsh-beer-theme](https://github.com/tcnksm/oh-my-zsh-beer-theme)
